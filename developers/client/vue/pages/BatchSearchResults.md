# BatchSearchResults

> This page will list all the results of a batch search.

## Props

| Prop name | Description                             | Type          | Values | Default |
| --------- | --------------------------------------- | ------------- | ------ | ------- |
| uuid      | The unique id of the batch search       | string        | -      |         |
| indices   | The indices of the current batch search | string\|array | -      |         |
