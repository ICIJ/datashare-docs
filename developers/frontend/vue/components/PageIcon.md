# PageIcon

> A squared icon to display on every page title

## Props

| Prop name | Description                     | Type                  | Values | Default |
| --------- | ------------------------------- | --------------------- | ------ | ------- |
| icon      | A FortAwesome definition object | string\|array\|object | -      |         |
| size      | A FortAwesome size string       | string                | -      | '2x'    |
